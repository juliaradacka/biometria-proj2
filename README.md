# Biometria – Projekt 1

## Opis
Projekt realizowany w ramach przedmiotu **Biometria**.  
Aplikacja służy do edycji i analizy obrazów rastrowych.

## Dostępne funkcje

### Operacje punktowe
- Negatyw
- Wyrównanie histogramu (auto-korekta)
- Korekta jasności (parametryczna)
- Korekta kontrastu (parametryczna)
- Binaryzacja (parametryczna)
- Konwersja do skali szarości:
  - Luminancja (oko ludzkie)
  - Średnia RGB
  - Jasność (rozstęp)

### Filtry splotowe
- Filtr uśredniający
- Filtr Gaussa
- Wyostrzanie
- Wykrywanie krawędzi (Sobela)
- Wykrywanie krawędzi (Roberts)
- Filtr własny (konfigurowalna maska)

### Operacje morfologiczne
Parametry: możliwość wyboru **rozmiaru** i **kształtu** elementu strukturalnego (SE); dla **erozji** i **dylatacji** dodatkowo wybór **liczby iteracji**.
- Erozja
- Dylatacja
- Otwarcie
- Zamknięcie

## Uruchomienie
Aby uruchomić aplikację, wystarczy uruchomić klasę **`Main`** na branchu **`develop`**.
